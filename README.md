# AI Powered Open Pit Mining Analytics Dashboard

An AI-powered mining analytics dashboard built using:

- Streamlit
- Plotly
- Random Forest
- XGBoost
- Scikit-Learn

## Features

- Interactive 3D Ore Body Viewer
- Heatmaps
- KPI Dashboard
- Feature Importance Analysis
- Ore Grade Prediction
- Revenue & Profit Estimation

## Run

```bash
pip install -r requirements.txt
streamlit run app/app.py
```
